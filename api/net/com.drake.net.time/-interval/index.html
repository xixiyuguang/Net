    net / com.drake.net.time / Interval

    Interval¶

    class Interval :Serializable

    轮循器

    操作

    1. 开启: 只有在闲置状态下才可以开始
    2. 停止
    3. 暂停
    4. 继续
    5. 重置: 重置不会导致轮循器停止
    6. 开关: 开启|暂停切换
    7. 生命周期

    回调: 允许多次订阅同一个轮循器

    1. 每个事件
    2. 停止或者结束

    Parameters¶

    end - 结束值

    period - 事件间隔

    unit - 事件单位

    initialDelay - 第一次事件的间隔时间

    start - 开始值, 当start]比end值大, 且end不等于-1时, 即为倒计时
